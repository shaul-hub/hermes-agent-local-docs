# Company AI Resource Platform

기업/팀 단위에서 AI 도구, 모델, 프롬프트, 정책, 학습 자료, 도입 사례를 한 곳에서 탐색·관리·공유하기 위한 리소스 플랫폼입니다.

## 현재 상태

- 초기 조사 및 기획 문서 정리 완료
- 코드 구현 전 문서 중심 리포지터리
- GitHub 리포지터리는 사용자가 직접 생성 예정
